Introduction

This application demonstrates d3.js's ability to generate visualizations - in this case, a cartographic visualization - that display change over time.

Software Used

The software used to build this demo includes:
  • d3.js - to generate the visualization
  • colorbrewer.js - to to generate a colorblind-safe color palette
  • queue.js - to manage asynchronous data requests from the web client
  • and a little bit of jQuery to help manage the DOM

Data Sources

The data displayed by this demo come from:

  • United States Bureau of the Census
  • Wikipedia

Note

The observant user will note that some states were enumerated by the Census Bureau before they were admitted to the union. For example, Vermont was enumerated in 1790, even though it was not admitted to the union until 1791.
